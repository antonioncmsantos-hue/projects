# Data

The dataset is **not included** in this repository because it is hosted by a third party on Kaggle.

## Source (Kaggle)
Loan Default Dataset (author: yasserh)  
https://www.kaggle.com/datasets/yasserh/loan-default-dataset/data

## Expected local path
After downloading and extracting, place the CSV file in this folder with the exact name below:
- `data/Loan_Default.csv`

## Download (manual)
1. Open the Kaggle link above.
2. Download the dataset.
3. Extract the files.
4. Rename the main CSV to `Loan_Default.csv` (if needed).
5. Move it into this folder: `data/`
